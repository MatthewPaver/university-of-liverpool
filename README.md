- **COMP122 - Object-Oriented Programming** ‚Äî Java programming, OOP principles, design patterns
- **COMP124 - Computer Systems** ‚Äî Computer architecture, operating systems, networking fundamentals

- Introduction to AI and machine learning concepts
- Computer systems and architecture
